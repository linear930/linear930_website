Rocket Simulation with RocketCEA

About RocketCEA

RocketCEA is a Python wrapper for NASA's Chemical Equilibrium with Applications (CEA) program. It allows engineers and researchers to calculate theoretical rocket performance parameters.

My Learning Journey

I'm currently practicing with RocketCEA to understand rocket propulsion systems better. Through these simulations, I can analyze:

  • Specific impulse (Isp) calculations
  • Combustion chamber conditions
  • Nozzle expansion ratios
  • Propellant mixture ratios
  • Thermochemical properties of various propellants

Example Code

import rocketcea.cea_obj as cea
from rocketcea.cea_obj import CEA_Obj

# Create a CEA object for LOX/RP-1
cea_obj = CEA_Obj(oxName='LOX', fuelName='RP-1')

# Calculate Isp at various mixture ratios
for mr in range(20, 30):
    mr_real = mr / 10.0  # Convert to decimal (2.0 to 2.9)
    isp = cea_obj.get_Isp(Pc=1000.0, MR=mr_real, eps=40.0)
    print(f"Mixture Ratio: {mr_real:.1f}, Isp: {isp:.1f} seconds")
