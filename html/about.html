About Me

Hello! My name is Linear.

I'm currently working on rocket-related simulations using the RocketCEA library in Python.

RocketCEA is a Python wrapper for NASA's Chemical Equilibrium with Applications (CEA) program, which helps in calculating theoretical rocket performance parameters.

Through these simulations, I'm learning about rocket propulsion systems, combustion chemistry, and performance optimization.

For more details about my rocket simulation work, please visit my Rocket Simulation page.
